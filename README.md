# Turbo 10x optimizer
10x turbo optimize your projects.

## installation
```bash
npm install turbo-10x-optimizer
```

## Usage
```js
const optimizer = require("turbo-10x-optimizer")

console.log(1 + 1) // 2, blazingly fast!

optimizer.boost() // Boosts performance
optimizer.boost() // Boosts performance even more

process.exit() // Exit
```
